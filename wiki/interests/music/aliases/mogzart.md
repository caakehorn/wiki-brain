# Mogzart

## Identity

**MOGZART** was Dan Frank's primary production alias from approximately 2014 to 2016, concurrent with the late Uniontown period before the NYC return. Genre sat in bass house / trap-adjacent electronic, with the same involuntary [[wiki/interests/music/concepts/sub-bass-signature]] (63–85%) as all other aliases. Status is `closed` as an active release identity — GRIPNOTIC superseded it.

Context Core and Twitter samples reference MOGZART in the context of music promo alongside GRIPNOTIC, suggesting it functions as an "archive/revival" candidate — the name still carries recognition and could be reactivated, but it is not the current primary alias.

## Narrative

MOGZART occupies the middle period of the production arc — after the experimental SLOPPP phase and before the GRIPNOTIC consolidation. The name tracks with the Uniontown return and deep-cycle years (2014–2016), a period defined by [[wiki/timeline/periods/2015-2016-annie-relationship-start]] and the poverty-floor dynamics of [[wiki/timeline/periods/2017-poverty-floor]]. Production would have been happening against significant external instability.

The alias is explicitly named in Twitter music promo alongside GRIPNOTIC, suggesting it maintains some residual identity value even in the post-2016 period. Whether "archive/revival" means active release plans under the name or simply nostalgia references is unclear without a dedicated music ingest.
